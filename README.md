# Сайт репетитора по математике

Одностраничный статичный сайт на `HTML + CSS + JS` без CMS, админки, базы данных и авторизации. Текущая версия сделана под репетитора по математике и построена без формы: только прямые контакты через Telegram, WhatsApp, телефон и email.

## Структура проекта

```text
.
|-- index.html
|-- README.md
`-- assets
    |-- css
    |   `-- styles.css
    |-- img
    |   |-- favicon.svg
    |   |-- math-workspace.svg
    |   `-- og-cover.svg
    `-- js
        `-- main.js
```

## Запуск

### Быстрый вариант

Откройте `index.html` в браузере.

### Через локальный сервер

Из корня проекта:

```bash
python -m http.server 5500
```

Откройте `http://localhost:5500`.

## Что менять

### Тексты и блоки

Все основные тексты находятся в `index.html`.

Ключевые секции:

- `#hero` — первый экран
- `.contact-strip` — быстрые способы связи
- `#about` — о занятиях
- `#programs` — направления
- `#process` — как работает обучение
- `#reviews` — отзывы
- `#faq` — вопросы
- `#contacts` — блок контактов

### Контакты

Тоже меняются в `index.html`.

Замените:

- `https://t.me/mathlavrentieva`
- `https://wa.me/79851234567`
- `tel:+79851234567`
- `hello@mathlavrentieva.ru`

### Иллюстрации

Лежат в `assets/img/`.

- `math-workspace.svg` — главная тематическая графика
- `og-cover.svg` — картинка для Open Graph
- `favicon.svg` — иконка сайта

Если захотите поставить реальное фото преподавателя:

1. Добавьте файл в `assets/img/`
2. Замените путь в `index.html`
3. При необходимости обновите `og:image`

### Цвета и стиль

Меняются в `assets/css/styles.css`.

Основные переменные находятся в `:root`:

- `--color-bg`
- `--color-bg-alt`
- `--color-ink`
- `--color-primary`
- `--color-primary-dark`
- `--color-accent`

### Интерактив

В `assets/js/main.js` уже есть:

- мобильное меню
- анимации появления блоков
- FAQ-аккордеон
- модальное окно политики

Формы в текущей версии нет.

## Что уже реализовано

- полностью новый дизайн, не повторяющий предыдущую версию
- адаптивная верстка
- новый путь CTA: только «Связаться со мной»
- прямые контакты вместо формы
- современная нишевая графика под математику
- SEO-основа: `title`, `meta description`, Open Graph, favicon, JSON-LD

## Перед публикацией

- замените демо-контакты на реальные
- обновите политику конфиденциальности
- замените `https://example.com` в JSON-LD на реальный домен
- проверьте ссылки на Telegram, WhatsApp, телефон и email
